## Project Description

This project involves designing and developing a multi-page personal portfolio website using HTML. The purpose of the website is to present an individual’s profile, skills, projects, and contact information in a structured and professional format. The site is divided into separate pages to improve organization, navigation, and user experience.

## Project Objectives

- Demonstrate understanding of HTML structure and semantic elements

- Practice organizing files using a clear folder structure

- Build a navigable website with multiple linked pages

- Present personal information professionally online
## Technologies Used
- HTML5
- Web browser for testing
- Text editor (e.g., VS Code)
## Features
- Multi-Page Navigation — Separate pages (Home, About, Projects, Contact) connected through a navigation menu for clear structure.
- Responsive Layout — Adapts to different screen sizes (desktop, tablet, mobile).
- Semantic HTML Structure — Uses meaningful tags like <header>, <main>, <section>, <article>, and <footer> for accessibility and SEO.
- Profile Introduction Section — Displays name, photo, and short bio.
- Skills Section — Lists technical or professional competencies.
- Projects Showcase — Highlights completed or sample projects with descriptions.
- Contact Form — Allows visitors to send messages via input fields and submit button.
- Image Integration — Supports profile pictures and project screenshots.
- Consistent Design — Uniform layout, colors, and typography across all pages.
- Organized Folder Structure — Files grouped logically (HTML pages, images folder, assets).
- External Links Support — Links to social profiles or other websites.
- Accessibility Friendly — Includes alt text for images and readable structure.
## How to Run
1. Clone this repository
2. Open `index.html` in your browser
   OR
   Run `npm install` then `npm start`
